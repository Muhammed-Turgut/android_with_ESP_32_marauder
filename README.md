<h2 align = "start">About App</h1>
<h3 align = "start">
A native control interface was developed to enable serial communication between an Android device and ESP32 via USB OTG.
With AI integration, modules were designed to generate dynamic local web servers (Captive Portals) for social engineering testing.
Hardware-level data processing was implemented, along with a modern dashboard design on the mobile side using Jetpack Compose.

<h1 align = "star"> Architecture – MVVM</h1>
The application is built using the MVVM (Model–View–ViewModel) architecture to ensure a clean, scalable, and maintainable codebase.
Model
Handles the data layer, including hardware-level data processing from the ESP32, USB-Serial communication logic, and AI-related data handling.
View
Implemented with Jetpack Compose, responsible only for rendering the UI and reacting to state changes exposed by the ViewModel.
ViewModel
Acts as a bridge between the Model and the View.
Manages UI state, business logic, and data flow using reactive state management, ensuring the UI remains lifecycle-aware.
</h3>
